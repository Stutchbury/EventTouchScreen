◆ setDebounceInterval()

void EventTouchScreen::setDebounceInterval ( uint16_t  ms)
inline

Set the debounce interval in milliseconds.

Just like physical buttons, screens can 'bounce' as they are pressed or released. This is particularly prevelant in resistive touch panels which are generally much slower to respond and require an 'assertive' touch.

The default bounce interval is 15ms but you will likely need to increase this for resistive panels. I have found 30-40ms works for mine.

Parameters
ms
